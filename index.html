Project Overview

This project analyses a video game sales dataset to explore trends across genres, platforms, and regions. The aim of the project was to use real-world data to apply descriptive statistics, visual analysis, and a basic machine learning approach.

Dataset Used

The dataset was sourced from Kaggle and contains information on over 16,000 video games. It includes details such as game name, platform, year of release, genre, publisher, and sales figures for North America, Europe, Japan, and global sales.

Dataset link: Video Game Sales Dataset (Kaggle)

Data Analysis

The data was analysed using Excel. Descriptive statistics were calculated to understand the distribution of global sales. Missing values were checked and found to be minimal, meaning the dataset required very little cleaning.

Visual Analysis

  • Total Global Sales by Genre
  • NA Sales vs Global Sales (scatter plot)
  • Number of Games per Platform

These charts helped identify patterns in sales performance and highlighted the importance of certain genres, platforms, and regions.

Machine Learning

A simple linear regression model was used to explore whether global sales could be predicted using North American sales. The results showed a clear positive relationship, supporting the findings from the visual analysis.

Conclusion

This project demonstrated how data analysis and basic machine learning can be applied to real-world datasets. The findings provided useful insights into the video game industry while meeting all requirements of the CA2 project.
